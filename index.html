Как установить

  1. Скачай нужный файл пресета выше
  2. Открой SillyTavern и перейди в AI Response Configuration
  3. Нажми Import Preset и выбери файл .json
  4. Для регексов: Extensions → Regex → Import
  5. Для скриптов: сначала установи JS-Slash-Runner, затем импортируй JSON через него
  6. Наслаждайся
